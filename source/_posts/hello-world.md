## Quick Start

### Create a new post

```bash
$ hexo new "My New Post"
```

```bash
$ hexo server
```

### Generate static files

```bash
$ hexo generate
```

More info: [Generating](https://hexo.io/docs/generating.html)

### Deploy to remote sites

```bash
$ hexo deploy
```

More info: [Deployment](https://hexo.io/docs/one-command-deployment.html)
